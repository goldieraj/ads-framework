## How to Use This Database

### Monthly Check-In Protocol
1. Visit the Ad Library for each Tier 1 brand (use Page ID for direct access)
2. Filter by "Images" media type, sort by "Total Impressions"
3. Look for NEW creatives that weren't there last month
4. Extract any new copy angles, hooks, or image styles
5. Add findings to the appropriate vault (copy, visual swipe, or hook library)

## Scaling Patterns Observed

| Pattern | Brands Using It | Description |
|---|---|---|
| Multi-persona | RejuvaCare, BioRoot Labs | Same product advertised under 3-5 different "people" (doctor, patient, therapist) |
| Multi-page | Rosabella, Alevia, Mira | Same product/copy run from 4-10 different Facebook pages |
| Creative reuse | All brands | 6-14 ads using identical creative with different copy |
| Long-form primary text | RejuvaCare, BioRoot, Alevia | 1,500-3,000 word advertorials IN the ad copy itself |
| Affiliate model | RejuvaCare, Rosabella, Alevia | affId tracking in URLs, multiple affiliates running same product |
| Geo-targeting | RejuvaCare, Gadgets Club | URL parameters indicate geographic targeting tests |
| Landing page split testing | RejuvaCare | v6, v9, v10, sa variants in URLs |
