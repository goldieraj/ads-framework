## KATRINA PAHOMOVA — MASTER PATTERN ANALYSIS

### Her Hook Formula Hierarchy (by performance):
1. **Statement + Binary** (1.1M, 439K, 42K): "Your hook decides if people scroll… or stay"
2. **Statement + Promise** (510K): "3 simple mask hooks you can steal and use"
3. **Benefit-First Statement** (112K): "The right font = instant personality"
4. **Relatability + Confession** (20.7K): "Too lazy to edit? Same"
5. **Question-Based** (25K): "Want to create carousel posts like these?" — LOWEST performer

### Visual Hook Patterns:
1. 3D puffy/inflatable text (HOOK, IDEA, FLUFF, GLASS, PUFF)
2. Physical movement toward camera (walking, jumping)
3. Mask/reveal transitions
4. Mixed typography (serif + sans-serif + script in same frame)
5. Pink/feminine color palette throughout

### Content Structure:
- Series format ("part 4") for return viewers
- Tutorial disguised as hook demonstration
- Prompt sharing (gives away the "secret")
- Tool recommendations (lower engagement)

### Key Lessons for Ads:
1. Statement hooks MASSIVELY outperform question hooks
2. Physical movement in first frame stops scroll
3. 3D text overlays create visual novelty
4. "Part X" series format builds anticipation
5. Giving away the formula/secret IS the hook
6. Binary choices create tension ("scroll or stay", "views or crickets")
