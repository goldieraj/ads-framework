# The Feedback System: How to Get Smarter With Every Dollar

This folder contains the `FEEDBACK_DATABASE_TEMPLATE.csv`. This is the most important file in your entire business.

If you test 100 ads and don't log the data, you are gambling. If you test 100 ads and log the hook, angle, format, and outcome of each one, you are building an intelligence asset that guarantees future profitability.

## How to Use the Database

Every time you launch a batch of ads, you must log them in the CSV. After 48-72 hours, pull the data from Meta Ads Manager and fill in the metrics.

Then, you run the **Winner/Loser Autopsy** (using Master Prompt #5) to fill in the final two columns: `Outcome_Decision` and `Diagnosis_Lesson`.

### The Metrics That Matter

1. **Hook Rate (3-second plays / Impressions):** Target is >25%. If it's lower, your first 3 seconds are boring. The angle might be good, but nobody is staying to hear it.
2. **Hold Rate (ThruPlays / 3-second plays):** Target is >10%. If Hook Rate is high but Hold Rate is low, you used clickbait. The hook didn't match the body copy.
3. **CTR (Link Clicks / Impressions):** Target is >1%. If Hook and Hold are good but CTR is low, your Call to Action is weak or the offer isn't compelling.
4. **CPA / ROAS:** The ultimate truth. If CPA is profitable, scale it.

### The Feedback Loop

When Hermes (your creative strategist agent) builds your next batch of ads, you will feed this CSV into its context window. 

Hermes will read the `Diagnosis_Lesson` column and say: *"I see that for this product, Contrarian hooks are getting a 30% Hook Rate, but Authority Personas are getting the best ROAS. I will generate a new batch combining Contrarian hooks with Authority personas."*

That is how you build a money machine.
